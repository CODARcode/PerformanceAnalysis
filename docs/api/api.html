Classes¶

General¶

  • ExecData: event, communication, function and execution data structure

  • AnomalyStat: statistic manager of the detected anomalies

Anomaly Detection¶

  • ADDefine: general definition, constants, error codes and etc.

  • ADParser: parser

  • ADEvent: event manager

  • ADOutlier: anomaly detection algorithm

  • ADio: I/O handler

Parameters¶

  • ParamInterface: abstract class of paramters of anomaly detection algorithms

  • SstdParam: parameters for statistic analysis-based anomaly detection algorithm

Network¶

  • NetInterface: abstract class of network layer

  • MPINet: MPI-based network layer

  • ZMQNet: ZeroMQ-based network layer

  • Message: message class used for communication

Utils¶

  • RunStats: class to compute running statistics

  • RunMetric: class to compute running metrics (derived from RunStats)

  • threadPool: thread pool

  • mtQueue: queue data structure for multi-threading support

  • DispatchQueue: dispatch queue with multi-threads
